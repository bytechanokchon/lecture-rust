### Comments
เป็นสิ่งที่เราใช้ในการเขียนอธิบายชุดคำสั่งต่าง ๆ (ไว้ใช้ในกรณีที่ชุดคำสั่งมีความซับซ้อน) โดยคอมไพเลอร์จะไม่สนใจคอมเมนท์

    // hello, world

เราจะใช้สัญลักษณ์ `//` ในการเขียนคอมเมนท์

    // So we're doing something complicated here, long enough that we need
    // multiple lines of comments to do it! Whew! Hopefully, this comment will
    // explain what's going on.

ตัวอย่างการคอมเมนท์ที่ดี

    fn main() {
        // I'm feeling lucky today
        let lucky_number: i16 = 7;
    }
