### Installation
1. ติดตั้ง rustup [download](https://rust-lang.org/tools/install/)
2. ติดตั้งตามขั้นตอน
3. ใช้คำสั่ง rustc --version, ควรมีเลขเวอร์ชันของ rust ขึ้นมา

*rustup* เป็นเครื่องมือติดตั้ง rust และช่วยจัดการเวอร์ชัน rust และเครื่องมือที่เกี่ยวข้อง

[คู่มือการติดตั้งฉบับเต็ม](https://doc.rust-lang.org/book/ch01-01-installation.html)
