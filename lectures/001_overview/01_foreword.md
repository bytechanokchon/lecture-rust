### Foreword
ภาษา rust พัฒนามาไกลมากในช่วงเวลาไม่กี่ปีที่ผ่านมา จากการสร้างของกลุ่มผู้ที่สนใจเพียงไม่กี่คน กลายเป็นภาษาโปรแกรมที่ได้รับความนิยมและเป็นที่ต้องการมากที่สุดในโลก เมื่อมองย้อนกลับไป เป็นเรื่องที่ปฏิเสธไม่ได้ว่าศักยภาพของ rust สามารถดึงดูดความสนใจและเข้ามามีบทบาทในวงการเขียนโปรแกรม และเปลี่ยนแปรงอุตสาหกรรมต่าง ๆ อย่างกว้างขวาง

### เครื่องมือที่ใช้ร่วมกับ rust
1. **Cargo** เครื่องมือจัดการโปรเจกต์ (build tool) และระบบจัดการไลบรารี่/แพ็คเกจ ทำหน้าที่ช่วยนักพัฒนาสร้างโปรเจกต์, จัดการ dependency (ไลบรารี่ภายนอก), build โค้ด, run และ test
2. **rustfmt** เครื่องมือจัดการรูปแบบ (format) ของภาษา rust ให้เป็นระเบียบและตามาตรฐานและสไตล์ของชุมชน rust
3. **Rust language server** ช่วยนักพัฒนาในการทำงานร่วมกับภาษา rust

ด้วยเครื่องมือเหล่านี้ ทำให้ระบยบนิเวศของ rust สามารถทำงานได้อย่างมีประสิทธิภาพ
