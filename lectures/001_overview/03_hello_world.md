### Hello World
ตามธรรมเนียมในโลกของการเขียนโปรแกรม โปรแกรมแรกของเรามักจะเป็น hello world เสมอ 

#### การตั้งค่าที่อยู่ไฟล์
เราจะสร้างโฟลเดอร์สำหรับจัดเก็บไฟล์ โดยจะตั้งชื่อโฟลเดอร์ว่าอะไรก็ได้

โดยการเขียน rust program เราจะต้องตั้งไฟล์ที่นามสกุลไฟล์ลงท้ายด้วย .rs เสมอ เช่น main.rs และหากชื่อไฟล์มีหลายคำ จะขั้นด้วย _ เสมอ เช่น hello_world.rs

เมื่อเราสร้างไฟล์เสร็จ โปรแกรมแรกของเราจะเขียนชุดคำสั่งดังนี้

    fn main() {
        println!("Hello, World!");
    }

เมื่อเราเขียนชุดคำสั่งเรียบร้อยแล้ว เราสามารถรันโปรแกรมได้ด้วยขั้นตอนดังนี้
1. ใช้คำสั่ง rustc main.rs (แปรงชุดคำสั่ง rust เป็นภาษาเครื่อง)
2. เรียกใช้งานไฟล์ที่ถูกแปรงเป็นภาษาเครื่องแล้ว เช่น main.exe

#### โครงสร้างของโปรแกรม
    fn main() {
        
    }

ชุดคำสั่งด้านบนเรียกว่าฟังก์ชันเมน ซึ่งเป็นฟังก์ชันพิเศษ เพราะมันเป็นชุดคำสั่งแรกที่จะถูกเรียกใช้งานโดย compiler

    println!("Hello, World!");

ชุดคำสั่งนี้ทำหน้าที่แสดงข้อความไปที่หน้าจอ โดยมีรายละเอียดสำคัญดังนี้

println! เป็นการเรียก **มาโคร (macro)** ของภาษา rust โดยหากเป็นฟังก์ชันปกติจะเขียนเป็น println (ไม่มีเครื่องหมาย !) *เราจะเรียนรู้เกี่ยวกับมาโครในภายหลัง ตอนนี้จำไว้ว่ามาโครเป็นฟังก์ชันพิเศษ*

ต่อมา เราปิดท้ายด้วยเครื่องหมายเซมิโคลอน (;) ซึ่งเป็นเครื่องหมายที่ใช้บอกว่า ชุดคำสั่งบรรทัดนี้สิ้นสุดลงแล้ว และสามารถดำเนินการคำสั่งถัดไปได้เลย

#### การคอมไพล์และการดำเนินการชุดคำสั่ง (compilation and execute)
เมื่อเราเขียนโปรแกรมเสร็จ ก่อนที่เราจะรันโปรแกรมของเรา เราจำเป็นต้องคอมไพล์มันโดยใช้คอมไพเลอร์ของ rust ก่อนเสมอ โดยใช้คำสั่ง rustc <file_name> 

    rustc main.rs

เพื่อคอมไพเลอร์ทำการแปรงชุดคำสั่งเป็นภาษาเครื่องเรียบร้อยแล้ว ในไฟล์เดอร์จะแสดงไฟล์ที่จัดเก็บชุดำสั่งของเราในรูปแบบภาษาเรื่อง เช่น main.exe

โดยหากเป็นโปรแกรมเล็ก ๆ การคอมไพล์ด้วยคำสั่ง rustc ก็เพียงพอแล้ว แต่ในโลกของการทำงานจริง โปรเจคขนาดใหญ่มักมีโปรแกรมที่ขนาดใหญ่ตาม ทำให้เราต้องการอุปกรณ์ที่ช่วยในการจัดการโปรแกรมของเราได้อย่างง่ายดาย เราจึงจำเป็นต้องใช้อุปกรณ์ที่ชื่อว่า Cargo ซึ่งช่วยในการจัดการโปรเจค Rust ที่ใช้งานจริง
